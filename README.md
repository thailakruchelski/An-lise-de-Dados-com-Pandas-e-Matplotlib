# An-lise-de-Dados-com-Pandas-e-Matplotlib
Gráfico de linha:
Mostra a variação dos valores ao longo do dataset, permitindo identificar tendências de aumento ou queda. É útil para perceber padrões contínuos nos dados. Também ajuda a visualizar possíveis picos ou quedas bruscas nos valores.
Gráfico de barras:
Evidencia quais categorias aparecem com maior frequência. Facilita a comparação entre diferentes grupos, destacando os mais representativos. Permite identificar rapidamente quais elementos são mais comuns no conjunto de dados.
Histograma:
Apresenta a distribuição dos valores numéricos, mostrando como eles se concentram. Ajuda a identificar a faixa de valores mais comum e a dispersão dos dados. Também permite observar a presença de valores extremos (muito altos ou baixos).
